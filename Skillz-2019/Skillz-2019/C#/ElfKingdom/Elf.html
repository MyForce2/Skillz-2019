class Elf

package ElfKingdom

extends GameObject › MapObject › BaseObject

This class represents a elf. Elves are actors that can build buildings. They can also move and attack other objects that are on the map.

The most important API is:
  • Attack
  • BuildPortal
  • CanBuildPortal

Orders the elf to build portal. The portal will be built on the location the elf is currently at.

CanBuildPortal() : bool

Returns whether the elf can build a portal. This method returns false if either: - The player doesn't have enough mana to build - The elf is too close to an existing building - The elf is not alive @returns Whether the elf can build a portal.
