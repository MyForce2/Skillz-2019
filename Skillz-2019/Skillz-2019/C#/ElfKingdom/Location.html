class Location

The most important API is:

  • Distance
  • InRange
  • Subtract

Towards(other:MapObject, length:int) : Location

Calculates the final location if the object moves towards location "other" with distance "length".

For example, calling location1.Towards(location2, 200) returns the point that is on the line between location1 and location2, at distance 200 from location1.

Numeric example:

location1 = new Location(0, 0)

location2 = new Location(1000, 0)

location1.towards(location2, 400) => Location(400, 0)

For more information and a more detailed example, see helper file on the Location API (under Info Pages on the sidebar).

Parameters:

other

The location to go towards.

length

The distance to travel.

Returns:

The final location.
